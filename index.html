Web Development

Enlistment Scheduler

This small web application made using React makes enlistment and scheduling a lot easier for all Ateneo students. Students can enter their classes that they want to choose from and select those that they want while the application automatically generates the schedules in real time.

TakBlue

TakBlue is a small web application wherein Ateneo students can see extra information from their AISIS schedules. It generates a small trail for their daily schedule that also contains the distances between two classes and the time it takes to go from one to another, given an average walking speed.
